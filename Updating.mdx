## Use automatic updates

You can turn on automatic updates to make sure that your system is updated at all times. Systems will always come in the form of a package.

<Danger>
  Deleting the package link will stop you from being able to update until you download the model again.
</Danger>

### Check if it's automatically updating

If your package is already automatically updating, you will see a <Icon icon="link-2" /> <Tooltip tip="">link</Tooltip> icon with a filled center. Hovering will warn you that it is automatically updating.

![Screenshot showing auto-updates toggled on](/images/updateson.png)

### Toggling auto-updates

If you find an icon of a link without a filled center <Icon icon="unlink-2" /> <Tooltip tip="">link</Tooltip>, you can toggle automatic updates by checking the `AutoUpdate` property under the PackageLink.

![Toggleautoupdates Pn](/images/toggleautoupdates.png)

## Manually Update

You can choose to manually update, instead.

### Checking if an update is available

If an update is available, a download icon <Icon icon="arrow-down" /> will show.

![Availableupdate Pn](/images/availableupdate.png)

### Updating Manually

To update when one is available, right click the package and select `Get Latest Package.`

![Screenshot showing right clicking the package and clicking Get Latest Package](/images/downloadnewversion.png)
